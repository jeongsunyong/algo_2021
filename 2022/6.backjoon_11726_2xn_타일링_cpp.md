## [2×n 타일링](https://www.acmicpc.net/problem/11726)
* 소요 시간 : 약 2시간

### 성공 코드
```cpp
#include <iostream>

using namespace std;

int main(void) {
	int n;
	cin >> n;
	long long * cnts = new long long[n];
	cnts[0] = 1;
	cnts[1] = 2;
	for (int i = 2; i < n; i++) {
		cnts[i] = (cnts[i - 1] + cnts[i - 2])%10007;
	}
	cout << cnts[n - 1] << endl;
	return 0;
}

```
